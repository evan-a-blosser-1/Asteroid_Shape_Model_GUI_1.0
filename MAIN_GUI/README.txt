HOW TO RUN

	1) Download MAIN_GUI

	2) Open this within your favortie Interactive Development Environment (IDE)
		- I used Visual Studio Code
	
	3) Make sure the following folders are in the working directory 
		(they should in the MAIN_GUI download)
		
		-Asteroid_CM
		-Asteroid_Info
		-OBJ_Files
		-Other_Info

	4) Then simply run Asteroids_In_MASCON1.py
